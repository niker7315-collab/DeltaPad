# PADFORGE — сборка портативного .exe

## Вариант A (ничего не ставить на свой ПК) — сборка в облаке через GitHub Actions

1. Зайди на https://github.com и создай бесплатный аккаунт (если его ещё нет).
2. Нажми "New repository" — назови как хочешь, например `padforge`, оставь Public, "Create repository".
3. На странице пустого репозитория нажми "uploading an existing file" (или "Add file" → "Upload files").
4. Перетащи туда **все файлы и папки из этого архива** — включая скрытую папку `.github`
   (в проводнике Windows её видно, просто зайди внутрь папки и выдели всё содержимое,
   включая `.github`, `main.js`, `preload.js`, `index.html`, `package.json`).
5. Нажми "Commit changes".
6. Перейди на вкладку **Actions** в репозитории — там сама запустится сборка
   (иконка с жёлтым кружком → зелёная галочка, минуты 2-3).
7. Когда сборка позеленеет — открой её, внизу будет раздел **Artifacts**,
   там файл `PADFORGE-exe` — скачай его (это zip с готовым .exe внутри).
8. Распакуй — внутри `PADFORGE-portable.exe`, готов к раздаче кому угодно.

Готово — ты не устанавливал ни Node.js, ни Electron, всё собралось на серверах GitHub.

## Вариант B — собрать локально (если есть Node.js)

## Что это
Тот же соунд-пад, но как настоящее оконное приложение для Windows, не в браузере.
Хранит пады и звуки в папке `padforge-data` рядом с exe-файлом — можно кидать
на флешку и переносить на другой ПК со всеми звуками.

## Один раз — собрать .exe

Тебе понадобится Node.js (скачать с https://nodejs.org, версия LTS) —
это нужно только для сборки. Пользователю готового .exe ставить уже
ничего не придётся.

Открой командную строку в этой папке и выполни:

```
npm install
npm run dist
```

Через минуту-две в папке `dist` появится файл:

```
dist/PADFORGE-portable.exe
```

Это и есть твоё приложение. Можно:
- переименовать как угодно,
- перенести на любой ПК с Windows,
- запускать двойным кликом — установка не требуется.

## Проверить без сборки exe

Если просто хочешь посмотреть, что получилось, прямо сейчас:

```
npm install
npm start
```

Откроется то же приложение в окне (это тот же код, что попадёт в exe).

## Где хранятся звуки

Рядом с exe создаётся папка `padforge-data` — там лежат сами звуковые файлы
и список падов. Если переносишь exe на другой компьютер и хочешь сохранить
звуки — скопируй `padforge-data` вместе с ним (в ту же папку).
